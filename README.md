# Marketing Analytics & Budget Optimization Dashboard

An interactive **data-driven marketing analytics dashboard** built using **Python, Pandas, and Streamlit** to analyze campaign performance, track KPIs, and simulate budget allocation strategies.

**Live Demo:**
https://anjali-marketing-analytics-dashboard-3gyvct5jxsolv2mjuwcyyv.streamlit.app/


## Problem Statement

Marketing teams often struggle to answer:

* Which platform gives the best ROI?
* Where should we allocate budget?
* What happens if we increase/decrease spend?
* How much profit can we expect next month?

This dashboard provides **real-time insights and simulations** to make smarter budget decisions.



## What the Dashboard Explains

### Current Spend Overview

Shows:

* Total budget spent
* Total ROI (profit)

Gives a clear baseline of current performance.



### Campaign Performance Analysis

Displays:
* CTR(Click-Through Rate): Measures ad engagement
* CPC(Cost Per Click): Measures traffic acquisition cost
* CPA(Cost Per Acquisition): Measures customer acquisition efficiency
* ROI(Return on Investment): Measures overall campaign profitability
* Platform-wise comparisons
* Daily ROI trends

Helps identify:

* Best performing platforms
* Underperforming campaigns
* Trends over time



###  Budget Reallocation Recommendation

Simulates:

> "What if we move X% budget from worst â†’ best platform?"

Uses ROI efficiency to:

* compare platforms
* estimate additional profit
* recommend optimal reallocation

Outputs:

* Current ROI
* New ROI
* Expected gain



### Budget Scaling & Forecast

Answers:

> "If my total budget changes, what happens to conversions & profit?"

Allows:

* scaling spend up/down
* forecasting conversions
* estimating revenue & ROI

Useful for planning future campaigns.



### ROI Forecasting

Applies regression to:

* predict future ROI trends
* estimate next 30 days performance

Supports proactive decision-making.
